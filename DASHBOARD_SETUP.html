🚀 Method 1: Browser (Easiest)

Step 1: Make sure XAMPP is running (Apache + PostgreSQL)

Step 2: Visit the seeder script:

http://localhost/finance/admin/seed-dashboard-data.php

Step 3: The script will display progress and confirm when complete

Step 4: Visit the dashboard:

http://localhost/finance/admin/admin.php

✓ Charts should now display data!

🖥️ Method 2: Command Line (PowerShell)

Step 1: Open PowerShell and navigate to the project:

cd c:\xampp\htdocs\finance\admin

Step 2: Run the seeder with PHP CLI:

"C:\xampp\php\php.exe" seed-dashboard-data.php

Step 3: You should see confirmation output showing inserted data

🆘 Troubleshooting

Charts show "No Data Available"?

  • ✓ Make sure you ran the seed script successfully
  • ✓ Check browser console (F12) for JavaScript errors
  • ✓ Verify database connection in includes/connectDB.php

Error about missing columns?

  • ✓ Ensure database schema is installed: php database/install_schema.php
  • ✓ Verify PostgreSQL is running and connected
