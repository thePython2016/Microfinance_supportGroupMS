#  Microfinance Support Group Management System

A web-based **Microfinance Support Group Management System** integrated with **SMS functionality (G-Cash Point)** and an **AI Assistant** to manage member information, track financial activities, and improve communication through automated SMS notifications and smart assistance.

The system helps microfinance groups digitize operations such as member registration, savings tracking, loan management, and communication.

🔗 Live Demo: https://microfinance-support-management-system.onrender.com/


#  System Overview

This system allows microfinance organizations to:
- Manage member registration and profiles
- Track savings, loans, and repayments
- Send automated SMS notifications (payment updates, reminders)
- Provide AI-powered assistant for user support and system guidance
- Improve transparency and financial accountability
- Provide an easy-to-use admin dashboard



#  Key Features

#  How to Use the System

## 1. Login
- If you already have an account, go to the login page
- Enter your **username/email and password**
- Click **Login** to access the system dashboard

## 2. Register
- If you are a new user, click **Register**
- Fill in required details (name, contact, password, etc.)
- Submit the form to create an account
- After successful registration, login to access system features

## 3. System Access
After logging in, users can:
- View and manage member information
- Record financial transactions
- Monitor savings and loans
- Receive SMS alerts and notifications
- Use AI Assistant for help and system guidance

## 4. AI Assistant Features
- Ask questions about system usage
- Get instant support for financial operations
- Receive guidance on managing members and loans
- Improve decision-making with smart responses

#  Authentication Flow

1. User registers an account
2. Credentials are securely stored in PostgreSQL (Supabase)
3. User logs in using email/username and password
4. Session authentication is created
5. Logout destroys session securely
